# Verasight JSM Omnibus Survey

Public survey data files for Verasight JSM Omnibus Survey.

## Links

- Source report: <https://reports.verasight.io/reports/jsm-2025>
- Verasight Data Library: <https://data.verasight.io/>
- Data release index: <https://github.com/Verasight/verasight-data-releases>

## Survey

- Report id: `2025-091`
- Wave id: `2025-091`
- Field dates: 2025-08-13 to 2025-08-18
- Population: United States adults
- Questions: 24
- Base size: N=1000.

## Files

- `data/questions.csv`: question text, source URLs, and sample sizes
- `data/toplines.csv`: response options and aggregate results
- `data/codebook.csv`: response option labels
- `docs/methodology.md`: report-specific methodology summary
- `docs/questionnaire.md`: question wording
- `metadata/report.json`: report metadata
- `CITATION.cff`: citation metadata
- `REUSE.md`: reuse terms

## Reuse

These files contain aggregate survey outputs only. They do not include respondent-level data.

Contact Verasight for reuse, redistribution, adaptation, or commercial use.
